# chat-app

## Requirements
- python 2.7.9
- flask
- wtforms
- flask-wtf
- flask-sqlalchemy
- flask-socketio
- flask-login
    
## Instructions
- Edit config.py in app. Set username and password for mysql db.
- Create db chat-app in mysql.
- If using ssl configure in run.py
- To run:
    * with ssl    : python run.py ssl
    * without ssl : python run.py
